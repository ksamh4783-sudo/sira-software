# 🤝 المساهمة في سيرا للبرمجيات

شكراً لاهتمامك بالمساهمة في مشروع سيرا للبرمجيات! نرحب بجميع المساهمات.

## 🚀 كيفية المساهمة

### 1. إعداد البيئة التطويرية

```bash
# استنساخ المشروع
git clone https://github.com/ksamh4783-sudo/sira-softwere.git
cd sira-softwere

# تثبيت dependencies
npm install
cd backend && npm install && cd ..
```

### 2. إنشاء فرع جديد

```bash
git checkout -b feature/your-feature-name
```

### 3. إجراء التغييرات

- اكتب كود نظيف ومفهوم
- اتبع معايير الـ Coding Style الموجودة
- أضف تعليقات عند الحاجة

### 4. اختبار التغييرات

```bash
# بناء المشروع
npm run build

# تشغيل الاختبارات
npm test
```

### 5. إرسال التغييرات

```bash
git add .
git commit -m "feat: add your feature description"
git push origin feature/your-feature-name
```

### 6. إنشاء Pull Request

- اذهب إلى صفحة المشروع على GitHub
- اضغط على "New Pull Request"
- املأ الوصف بشكل واضح
- انتظر المراجعة

## 📋 معايير الكود

### TypeScript/React
- استخدم TypeScript بشكل صحيح
- اتبع مبدأ DRY (Don't Repeat Yourself)
- استخدم Components قابلة لإعادة الاستخدام

### CSS/Tailwind
- استخدم Tailwind CSS للتنسيق
- تجنب CSS المخصص قدر الإمكان
- اتبع نظام التصميم الموجود

### Backend
- استخدم async/await
- تعامل مع الأخطاء بشكل صحيح
- أضف تعليقات للـ API endpoints

## 🐛 الإبلاغ عن الأخطاء

عند الإبلاغ عن خطأ، يرجى تضمين:

- معلومات البيئة (OS, Browser, إلخ)
