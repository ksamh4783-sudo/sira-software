🚨 TOP 10 مشاكل + إصلاحات كاملة (AI Doctor Analysis)
1. Vite Security (CVE-2025-58751 - HIGH)
المشكلة: Vite 7.2.4 عرضة لـ path traversal عبر public dir symlinks.
Fix:
bash



npm update vite @vitejs/plugin-react --latest
في vite.config.ts:
ts



export default defineConfig({
  server: {
    fs: { strict: true },  // block symlinks
    headers: [{ key: 'X-Content-Type-Options', value: 'nosniff' }]
  }
})
2. localStorage Security (CRITICAL - Production Risk)
المشكلة: كل البيانات (routers, vouchers, users) في localStorage = insecure/cleartext.
Fix: Encrypt + migrate to IndexedDB + backend:
bash



npm i crypto-js idb-keyval
في localApi.ts:
ts



import CryptoJS from 'crypto-js';
const SECRET = 'your-32-char-secret-key-here!!!';
export const setData = (key: string, data: any) => {
  const encrypted = CryptoJS.AES.encrypt(JSON.stringify(data), SECRET).toString();
  localStorage.setItem(key, encrypted);
};
3. No Authentication/Session Management
المشكلة: AuthContext بسيط، no JWT/refresh tokens، demo creds hardcoded.
Fix: أضف real auth:
bash



npm i lucia-auth @lucia-auth/adapter-d1
في AuthContext.tsx:
tsx



const login = async (email: string, password: string) => {
  const res = await fetch('/api/auth', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email, password }),
    credentials: 'include'  // cookies
  });
};
4. Missing Error Boundaries + Loading States
المشكلة: Pages تتعطل بدون error handling.
Fix: أنشئ ErrorBoundary.tsx:
tsx



class ErrorBoundary extends React.Component {
  state = { hasError: false };
  static getDerivedStateFromError() { return { hasError: true }; }
  render() {
    return this.state.hasError ? <div>خطأ: إعادة تحميل...</div> : this.props.children;
  }
}
5. TypeScript Strict Mode Off
المشكلة: tsconfig.json غير strict.
Fix في tsconfig.json:
json



{
  "compilerOptions": {
    "strict": true,
    "noImplicitAny": true,
    "strictNullChecks": true
  }
}
